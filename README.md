## Transisi test developer

How to run project:

- php Dasar :
    - cd /var/www/html/Transisi-test/phpDasar
- Laravel Dasar :
    - composer install
    - composer update
    - npm install 
    - npm run dev
    - php artisan migrate:fresh --seed
    - php artisan serve

## Requirement code

- php --> PHP 8.1.1
- laravel --> Laravel v8.80.0 (PHP v8.1.1)
- mysql --> mysql  Ver 8.0.27-0ubuntu0.20.04.1 for Linux on x86_64 ((Ubuntu))
